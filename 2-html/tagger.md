# Vanlige HTML-tagger

## Overskrifter - headings

```HTML
<h1>My main title</h1>
<h2>My top level heading</h2>
<h3>My subheading</h3>
<h4>My sub-subheading</h4>
```

## Avsnitt - paragraph

```HTML
<p>Dette er et avsnitt</p>
```

## Lister

**Uordnet/punkter**

```HTML
<ul>
    <li>technologists</li>
    <li>thinkers</li>
    <li>builders</li>
</ul>
```

**Ordnet/med tall**

```HTML
<ol>
    <li>technologists</li>
    <li>thinkers</li>
    <li>builders</li>
</ol>
```

## Lenker

```HTML
<a href="https://www.vg.no">Mozilla Manifesto</a>
```

## Bilder

```HTML
<img src="bilder/mittBilde.png" alt="Mitt bilde">
```
